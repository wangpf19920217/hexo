## 开始
<!-- mome -->
### 创建项目

``` bash
$ hexo new "项目名称"
```

查看更多: [Writing](https://hexo.io/docs/writing.html)

### 开启服务

``` bash
$ hexo server
```

查看更多: [Server](https://hexo.io/docs/server.html)

### 生成静态文件 到public文件夹下

``` bash
$ hexo generate
```

查看更多: [Generating](https://hexo.io/docs/generating.html)

### 提交到git

``` bash
$ hexo deploy
```

查看更多: [Deployment](https://hexo.io/docs/deployment.html)
